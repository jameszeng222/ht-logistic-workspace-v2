# 文档处理技能

## 工作流

1. 根据扩展名调对应工具（目前支持 parse_pdf）
2. 总结类：分段落摘要 → 全局摘要，控制总长度
3. 问答类：先 kb_search 找相关片段，再基于片段回答，标注来源
4. 字段提取：先让用户确认字段列表，逐个抽取并给置信度（高/中/低）

## 规范

- 引用必带页码/段落定位
- 答案必须在原文可验证，不可编造
- 片段不足以下结论时明确说"原文未提供足够信息"
- 大文档先告知将处理多少页，避免静默长任务
